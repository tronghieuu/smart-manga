ECharts is open source from Baidu's commercial front-end data visualization team. Based on html5 Canvas, it is a pure Javascript chart library that provides intuitive, vivid, interactive, and personalized data visualization charts. Innovative features such as drag-and-drop recalculation, data view, and range roaming greatly enhance the user experience and give users the ability to mine and integrate data. Learn more

ECharts official website: http://echarts.baidu.com/
