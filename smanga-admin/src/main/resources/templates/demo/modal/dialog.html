Setting options

The related functions of the modal window can be realized by data binding or Javascript. If you use data binding, you can add data- to the element, such as data-backdrop="".

Name Type default value Description
backdrop boolean or string 'static' true Mask layer, or use 'static' to specify that the mask layer is not associated with closing the modal window
keyboard boolean true Exit the modal window when pressing Esc key
show boolean true After the initialization is complete, the modal window will be displayed
remote path false

recommended to use data binding, or use jQuery.load

Remote URL example: 

<a data-toggle="modal" href="remote.html" data-target="#modal">Click me</a>
