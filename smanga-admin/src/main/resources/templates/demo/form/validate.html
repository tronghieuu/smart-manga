Introduction to jQuery Validate

jquery.validate.js is an excellent jQuery form validation plugin. It has the following characteristics:

  • Simple installation
  • Over 20 built-in data verification methods
  • In-line error message
  • Extensible data verification method
  • Use the built-in metadata or plug-in options to specify your validation rules
  • Elegant interaction design

Official website: http://jqueryvalidation.org/
